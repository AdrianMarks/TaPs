# TaPs

Tips and Payments - IOTA POC Project

Swift iOS application for iPhones. 

** N.B. Very much work in progress. **

## Compatibility

TaPs has been developed for iOS 10.0 and above.

## Dependencies

IotaKit

AvatarCapture

SwiftKeychainWrapper
